# Introducción a la Programación

Repositorio de apoyo para el curso **Introducción a la Programación**, orientado a estudiantes que inician en el desarrollo de algoritmos y programación en **lenguaje C**.

El objetivo del curso es desarrollar habilidades para **analizar problemas, diseñar algoritmos y construir programas que permitan resolver problemas computacionales en áreas como matemáticas, física e ingeniería.**

---

# Objetivos del curso

Al finalizar el curso el estudiante será capaz de:

- Comprender los **conceptos fundamentales de computación**.
- Entender la **estructura básica de una computadora**.
- Diseñar **algoritmos para la resolución de problemas**.
- Utilizar **pseudocódigo y diagramas de flujo**.
- Implementar algoritmos utilizando **lenguaje de programación C**.

---

# Contenido del curso

## 1. Fundamentos de computación

- Sistemas de numeración
  - Binario
  - Hexadecimal
- Representación de información
  - ASCII
  - Complemento a dos
- Modelo de Von Neumann
- Administración básica del sistema operativo

---

## 2. Algoritmos y programación estructurada

- Características de los algoritmos
- Principios de programación estructurada
- Pseudocódigo
- Diagramas de flujo

---

## 3. Programación en lenguaje C

- Estructura de un programa
- Variables y constantes
- Tipos de datos
  - `char`
  - `int`
  - `float`
  - `double`
- Modificadores
  - `short`
  - `long`
  - `unsigned`
- Operadores
- Expresiones
- Entrada y salida (`printf`, `scanf`)

---

## 4. Estructuras de control

### Estructura secuencial

Ejecución de instrucciones en orden.

### Estructuras condicionales

- `if`
- `if / else`
- `switch`

### Estructuras repetitivas

- `while`
- `for`
- `do-while`

---

## 5. Funciones

- Definición de funciones
- Variables locales y globales
- Paso de parámetros
- Recursividad
- Modularización de programas

---

## 6. Arreglos y estructuras

- Arreglos unidimensionales
- Arreglos bidimensionales
- Manejo de matrices
- Definición de estructuras

---
